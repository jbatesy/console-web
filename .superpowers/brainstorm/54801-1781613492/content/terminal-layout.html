Terminal Layout Options

How should the N console panes be arranged?

▶ deploy-check ▶ tail-logs ▶ db-status

helloworld
$ 

A — Tabbed (top tabs)

One terminal fills the screen, tabs across the top switch between panes. Clean, familiar.

▶ deploy-check
$ echo helloworld
helloworld
▶ tail-logs
$ tail -f /var/log/app.log
waiting...

B — Split panes (tiled)

All N terminals visible simultaneously in a grid. No switching needed — best for monitoring multiple streams at once.

C — Sidebar + main panel

Pane list in a left sidebar, selected pane fills the main area. Scales well to many panes.
